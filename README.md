# yanc-cli

## usage

```sh
# installation
yarn add -D @yanc/cli@"github:aistyler/yanc-cli#refs/tags/v1.3.3"

# or on yarn 2+
yarn add -D @yanc/cli@"github:aistyler/yanc#workspace=@yanc/cli&sermver:^1.3.3"
```

## configutation

- package.json

```js
// properties and default value
"yanc": {
  // verbose output
  "verbose": false,
  // whether browser environment
  "browser": false
}
```
